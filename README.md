# ISL Real-Time AI Translator 🤟

## 📌 Overview
This project converts Indian Sign Language (ISL) gestures into text using deep learning.

## 🚀 Features
- Real-time sign detection
- CNN + Transformer model
- MediaPipe landmark extraction
- Text-to-Speech output

## 🛠 Tech Stack
- Python, PyTorch
- MediaPipe
- React (Frontend)

## ▶️ How to Run
```bash
pip install -r requirements.txt
python training/train.py
